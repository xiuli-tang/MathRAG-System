求函数\(f(x) = x^3\)在点\((1, 1)\)处的切线方程。

解：\(f'(x) = 3x^2\)，则\(f'(1) = 3\)，切线方程为\(y - 1 = 3(x - 1)\)，即\(y = 3x - 2\)。

已知函数\(f(x) = e^x + \ln x\)，求\(f'(x)\)。

解：\(f'(x) = e^x + \frac{1}{x}\)。

求曲线\(y = \sin x\)在\(x = \pi\)处的切线方程。

解：\(y' = \cos x\)，在\(x=\pi\)处，\(y' = \cos \pi = -1\)，对应的y值为\(\sin \pi = 0\)。切线方程为\(y - 0 = -1(x - \pi)\)，即\(y = -x + \pi\)。
