已知函数\(f(x) = x^3 - 3x + a\)，讨论方程\(f(x) = 0\)的实根个数。

解：求导得\(f'(x) = 3x^2 - 3\)，临界点\(x = \pm 1\)。极大值\(f(-1) = 2 + a\)，极小值\(f(1) = -2 + a\)。当\(f(-1) < 0\)时，即\(a < -2\)，方程有1个实根；当\(f(1) > 0\)，即\(a > 2\)，方程有1个实根；当\(a = -2\)或\(a = 2\)时，方程有2个实根；当\(-2 < a < 2\)时，方程有3个实根。

讨论方程\(e^x = ax\)的实根个数（\(a > 0\)）。

解：令\(f(x) = e^x - ax\)。当\(x→-∞\)时\(f(x)→+∞\)，当\(x→+∞\)时\(f(x)→+∞\)。求导得\(f'(x)=e^x - a\)，临界点\(x=\ln a\)。极小值\(f(\ln a)=a(1-\ln a)\)。当\(f(\ln a)<0\)即\(a>e\)时有2个零点；当\(f(\ln a)=0\)即\(a=e\)时有1个零点；当\(f(\ln a)>0\)即\(0<a<e\)时无零点。
