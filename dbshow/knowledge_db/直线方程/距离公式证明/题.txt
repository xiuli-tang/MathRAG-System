求证：两条平行直线 $Ax + By + C_1 = 0$ 与 $Ax + By + C_2 = 0$ 间的距离为$$d = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}.$$分析：两条平行直线间的距离即为这两条平行直线中的一条直线上的一点到另一条直线的距离。证明：在直线 $Ax + By + C_1 = 0$ 上任取一点 $P(x_0, y_0)$，点 $P(x_0, y_0)$ 到直线 $Ax + By + C_2 = 0$ 的距离就是这两条平行直线间的距离，即$$d = \frac{|Ax_0 + By_0 + C_2|}{\sqrt{A^2 + B^2}}.$$因为点 $P(x_0, y_0)$ 在直线 $Ax + By + C_1 = 0$ 上，所以 $Ax_0 + By_0 + C_1 = 0$，即 $Ax_0 + By_0 = -C_1$，因此$$d = \frac{|Ax_0 + By_0 + C_2|}{\sqrt{A^2 + B^2}} = \frac{|-C_1 + C_2|}{\sqrt{A^2 + B^2}} = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}.$$
