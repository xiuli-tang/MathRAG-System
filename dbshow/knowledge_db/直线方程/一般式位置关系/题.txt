从一般式考虑:$l_1: A_1x + B_1y + C_1 = 0$，$l_2: A_2x + B_2y + C_2 = 0$$l_1 \perp l_2 \Leftrightarrow A_1A_2 + B_1B_2 = 0$$l_1 // l_2 \Leftrightarrow A_1B_2 - A_2B_1 = 0$ 且 $A_1C_2 - A_2C_1 \neq 0$ 或 $B_1C_2 - B_2C_1 \neq 0$，记忆式 $(\frac{A_1}{A_2} = \frac{B_1}{B_2} \neq \frac{C_1}{C_2})$$l_1$ 与 $l_2$ 重合，$A_1B_2 - A_2B_1 = 0$，$A_1C_2 - A_2C_1 = 0$，$B_1C_2 - B_2C_1 = 0$于是与直线 $Ax + By + C = 0$ 平行的直线可以设为 $Ax + By + D = 0$；垂直的直线可以设为 $Bx - Ay + D = 0$.
