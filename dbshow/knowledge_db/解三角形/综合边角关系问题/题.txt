在△ABC中，角A、B、C所对的边分别为a、b、c。已知sin A:sin B:sin C=2:1:$\sqrt{2}$，b=$\sqrt{2}$。(1) 求a的值；(2) 求cos C的值；(3) 求sin(2C-$\frac{\pi}{6}$)的值。解析 (1) 由正弦定理$\frac{a}{\sin A}=\frac{b}{\sin B}$，得a=b·$\frac{\sin A}{\sin B}$，因为sin A:sin B=2:1，b=$\sqrt{2}$，所以a=$\sqrt{2}$×$\frac{2}{1}$=2$\sqrt{2}$。(2) 由正弦定理可得a:b:c=sin A:sin B:sin C=2:1:$\sqrt{2}$，不妨设a=2k(k＞0)，则b=k，c=$\sqrt{2}$k，由余弦定理可得cos C=$\frac{a^{2}+b^{2}-c^{2}}{2ab}$=$\frac{(2k)^{2}+k^{2}-(\sqrt{2}k)^{2}}{2\times2k\times k}$=$\frac{3k^{2}}{4k^{2}}$=$\frac{3}{4}$。(3) 由题意结合(2)可知，sin C=$\sqrt{1-\cos^{2}C}$=$\sqrt{1-(\frac{3}{4})^{2}}$=$\frac{\sqrt{7}}{4}$，所以sin 2C=2sin Ccos C=2×$\frac{\sqrt{7}}{4}$×$\frac{3}{4}$=$\frac{3\sqrt{7}}{8}$，cos 2C=2cos^{2}C-1=2×($\frac{3}{4}$)^{2}-1=$\frac{1}{8}$，所以sin(2C-$\frac{\pi}{6}$)=sin 2Ccos $\frac{\pi}{6}$-cos 2Csin $\frac{\pi}{6}$=$\frac{3\sqrt{7}}{8}$×$\frac{\sqrt{3}}{2}$-$\frac{1}{8}$×$\frac{1}{2}$=$\frac{3\sqrt{21}-1}{16}$。

△ABC 的内角 A,B,C 的对边分别为 a,b, c. 已知△ABC 的面积为 $\frac{a^2}{3\sin A}$.(1) 求 sin Bsin C;(2) 若 6cos Bcos C=1, a=3, 求△ABC 的周长.说明  本题需要利用三角形面积公式构造等式, 进而获得第 (1) 问的结论; 第 (2) 问在原有已知条件的基础上提供了另两个等式, 使得周长可求.解析 (1) 由题设得 $\frac{1}{2}a\sin B = \frac{a^2}{3\sin A}$, 由正弦定理得 $\frac{1}{2}\sin C\sin B = \frac{\sin A}{3\sin A}$.故 sin Bsin C = $\frac{2}{3}$.(2) 由题设及 (1) 得 cos Bcos C - sin Bsin C = -$\frac{1}{2}$, 即 cos(B+C) = -$\frac{1}{2}$, 所以 B+C = $\frac{2\pi}{3}$, 故 A = $\frac{\pi}{3}$.方法一 由题设得 $\frac{1}{2}b\sin A = \frac{a^2}{3\sin A}$, 即 bc = 8.由余弦定理得 $b^2 + c^2 - bc = 9$, 即 $(b+c)^2 - 3bc = 9$, 得 b+c = $\sqrt{33}$.故△ABC 的周长为 a+b+c = 3 + $\sqrt{33}$.方法二 因为 a=3, 所以 2R = $\frac{a}{\sin A}$ = $2\sqrt{3}$ (R 为△ABC 外接圆的半径), 所以 sin Bsin C = $\frac{b}{2R} \cdot \frac{c}{2R}$ = $\frac{bc}{(2\sqrt{3})^2}$ = $\frac{bc}{12}$ = $\frac{2}{3}$, 则 bc = 8.由余弦定理得 $b^2 + c^2 - 2bc \cdot \cos \frac{\pi}{3} = 9$, 即 $b^2 + c^2 - bc = 9$,所以 $(b+c)^2 - 3bc = 9$,所以 $(b+c)^2 = 9 + 3bc = 9 + 3 \times 8 = 33$, 故 b+c = $\sqrt{33}$.故△ABC 的周长为 a+b+c = 3 + $\sqrt{33}$.
