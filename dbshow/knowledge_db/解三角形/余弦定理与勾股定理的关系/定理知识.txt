若 $b^2 + c^2 > a^2$，根据余弦定理的推论可知 $\cos A = \frac{b^2 + c^2 - a^2}{2bc} > 0$，则 A 为锐角。若 $a^2 + c^2 > b^2$，$a^2 + b^2 > c^2$，同理可得 B, C 为锐角。所以当 $b^2 + c^2 > a^2$，$a^2 + c^2 > b^2$ 且 $a^2 + b^2 > c^2$ 时，△ABC 是锐角三角形。

若 $b^2 + c^2 < a^2$，根据余弦定理的推论可知 $\cos A = \frac{b^2 + c^2 - a^2}{2bc} < 0$，则 △ABC 是钝角三角形且 A 是钝角。同理可得：若 $a^2 + c^2 < b^2$，则 △ABC 是钝角三角形且 B 是钝角；若 $a^2 + b^2 < c^2$，则 △ABC 是钝角三角形且 C 是钝角。

若 $b^2 + c^2 = a^2$，根据余弦定理的推论可知 $\cos A = \frac{b^2 + c^2 - a^2}{2bc} = 0$，则 △ABC 是直角三角形且 A 是直角。同理可得：若 $a^2 + c^2 = b^2$，则 △ABC 是直角三角形且 B 是直角；若 $a^2 + b^2 = c^2$，则 △ABC 是直角三角形且 C 是直角。综上可知，余弦定理是勾股定理的推广，而勾股定理是余弦定理的特例。
