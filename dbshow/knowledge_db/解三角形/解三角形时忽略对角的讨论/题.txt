在△ABC 中, a=$\sqrt{3}$, b=$\sqrt{2}$, B=45°, 求角A, C和边c.$\checkmark$ 正解 $>$ 由正弦定理 $\frac{a}{\sin A} = \frac{b}{\sin B}$, 得 $\frac{\sqrt{3}}{\sin A} = \frac{\sqrt{2}}{\sin 45°}$,$\therefore \sin A = \frac{\sqrt{3}}{2}$, 由 a > b 得 A = 60° 或 A = 120°.当 A = 60° 时, C = 180° - 45° - 60° = 75°.$\because \sin 75° = \sin(45° + 30°)$$= \sin 45° \cos 30° + \cos 45° \sin 30°$$= \frac{\sqrt{2} + \sqrt{6}}{4}$,$\therefore c = \frac{b \sin C}{\sin B} = \frac{\sqrt{2} + \sqrt{6}}{2}$.当 A = 120° 时, C = 180° - 45° - 120° = 15°.$\because \sin 15° = \sin(45° - 30°) = \sin 45° \cos 30° - \cos 45° \sin 30°$$= \frac{\sqrt{6} - \sqrt{2}}{4}$,$\therefore c = \frac{b \sin C}{\sin B} = \frac{\sqrt{6} - \sqrt{2}}{2}$.综上, A = 60°, C = 75°, c = $\frac{\sqrt{2} + \sqrt{6}}{2}$ 或 A = 120°, C = 15°, c = $\frac{\sqrt{6} - \sqrt{2}}{2}$.
